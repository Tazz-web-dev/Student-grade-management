# Student Grade Management System for Teachers

University Software Engineering course assignment (Project 8 of 8).

## Deliverables

1. **GitHub repo** with full source code (README, meaningful commits, organized structure).
2. **Software Development Document** (`.docx`) named by student ID number — follows `AssignmentSpecification.docx` formatting (A4, Times New Roman 12pt, 1.5 line spacing, chapter-based heading numbering, figure/table captions).

## Project scope

Project 8 — Student Grade Management System for Teachers. Key features:
- Roles: Teacher, Student, Admin
- Course & enrollment management with weighted grade categories
- Grade recording with weighted average calculation
- Student grade view (own grades, class averages anonymized)
- Reporting: distribution histogram, failing students, CSV export
- Audit log of grade changes
- Input validation (0–100 grade range), SQL injection prevention, role-based middleware

## Suggested tech stack (from manual)

- Backend: Python/Flask or Java/Spring Boot
- Database: MySQL or PostgreSQL
- Frontend: HTML/CSS/JS (React/Vue optional)
- Testing: JUnit or PyTest
